# /forge:c4 — diagramas C4 + overview navegável

Pré-requisito: `/forge:graph build` (rode `/forge:graph update` antes se tocou código). Em greenfield, rode após `/forge:design`; em feature, após `/forge:design`/`/forge:tasks` dos módulos tocados.

## Execução

```bash
bash .forge/scripts/c4.sh
```

Gera `.forge/graph/c4/{c1-context.mmd, c2-container.mmd, c3-component-<module>.mmd}` e `.forge/graph/overview.html`. O HTML reúne: os 3 níveis C4 renderizados (Mermaid), o índice de capabilities do baseline, e o estado/progresso dos changes ativos.

## Convenções (gate grep-negativo)

- **Sem pontos dentro de labels Mermaid** e **sem em-dash** — o gerador sanitiza (`money.ts` → `money ts`). Se você curar um `.mmd` à mão, mantenha a convenção.

## Curadoria (opcional)

Os diagramas saem da estrutura do grafo. Para enriquecer com intenção (rótulos de domínio, atores externos reais no C1), edite os `.mmd` ou invoque o `architecture-analyzer`. O `overview.html` referencia Mermaid via CDN (artefato de visualização em `.forge/graph/`, fora do commit — §20); abra no navegador.

## Regras

- `.forge/graph/c4/` e `overview.html` são gerados — não versione; regenere quando o grafo mudar.
